## Shipping

- [HudAim](https://github.com/8tp/hudaim) is a browser-based aim training platform with 6 game modes, replays, leaderboards, and anti-cheat. Live at [aim.chuds.dev](https://aim.chuds.dev/).
- [Coup](https://github.com/8tp/Coup) brings the bluffing card game online with real-time multiplayer, bots, and mobile-friendly rooms. Live at [coup.chuds.dev](https://coup.chuds.dev).
- [LiteStats](https://github.com/8tp/LiteStats) is a lightweight macOS menu bar app for CPU, RAM, storage, battery, and thermal monitoring.
- [chuds.dev](https://github.com/8tp/chuds.dev) is the portfolio and launchpad for the rest of the work.
